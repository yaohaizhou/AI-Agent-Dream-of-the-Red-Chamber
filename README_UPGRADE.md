# AI续写红楼梦项目 - 系统升级版

本项目实现了对原有AI续写红楼梦系统的全面升级，采用渐进式生成和高级质量检查技术，旨在生成与原著风格高度一致的续写内容。

## 升级亮点

### 1. 渐进式生成架构
- **分步生成**：回目 → 场景大纲 → 内容生成 → 诗词创作 → 整体润色
- **质量控制**：每步都有验证机制
- **古典风格**：严格遵循《红楼梦》的文学特色

### 2. 人物一致性保障
- **人物档案**：为宝玉、黛玉、宝钗等主要角色建立详细档案
- **多维度验证**：语言、行为、情感、关系四个维度一致性检查
- **动态调整**：根据人物设定调整生成内容

### 3. 高级质量评估
- **综合评分**：人物一致性、结构完整性、风格一致性多维度评估
- **智能反馈**：生成过程中实时质量监控
- **迭代优化**：未达标的生成内容自动优化

## 核心模块

### progressive_generator.py
实现渐进式生成流程，确保每个环节都符合古典文学标准。

### character_consistency_checker.py
提供高级人物一致性检查，确保角色行为符合原著设定。

### orchestrator.py (已更新)
集成新模块，协调整个生成流程。

## 使用方法

```bash
# 运行续写任务
python3 run_continuation.py
```

## 质量标准

- **回目格式**：八字对仗，符合原著风格
- **人物刻画**：性格、语言、行为与原著一致
- **文笔风格**：古典白话，贴近曹雪芹文风
- **诗词质量**：格律工整，意境优美
- **结构完整**：场景过渡自然，结尾留悬念
